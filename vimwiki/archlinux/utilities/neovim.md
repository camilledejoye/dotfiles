# Neovim

## Installation

For the stable release:
```sh
pacman -S neovim
```

For the nightly builds:
```sh
yay -S neovim-nightly
```

## Modules

Python module:
```sh
pacman -S python-pynvim
```

Python2 module (not sure it still required but just in case):
```sh
yay -S python2-pynvim
```

## Clipboard

```sh
pacman -S xclip
```
